<h2>Sobre este Repositório:</h2>

<p>Este repositório contém uma enciclopédia não-oficial e colaborativa dedicada à Parte 7 de JoJo's Bizarre Adventure — Steel Ball Run, obra do mangaká Hirohiko Araki. O objetivo é documentar, de forma organizada e acessível, todos os aspectos da narrativa: personagens, Stands, arcos, temas e o universo alternativo introduzido nesta parte.</p>

<h2> Funcionalidades:</h2>
<ul>
  <li>Intuito informativo</li>
  <li>Contagem Regressiva para o Lançamento do 1º Episódio</li>
</ul>
